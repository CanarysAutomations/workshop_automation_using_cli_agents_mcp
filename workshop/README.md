# Agents in SDLC Workshop

> **Automate the full software development lifecycle using GitHub Copilot CLI agents.**

## The Scenario

Your team maintains **Help Desk CRM** — a FastAPI ticketing system serving support teams. Using GitHub Copilot CLI, you will build a custom agent, implement a feature, generate CI/CD workflows, and deploy a self-healing system — all from the terminal.

---

## Prerequisites

| Requirement | Version |
|---|---|
| Python | 3.11+ |
| GitHub CLI (`gh`) | Latest |
| Git | 2.x |
| GitHub Copilot subscription | Individual / Business / Enterprise |
| OpenAI API key | Required for Exercise 5 only |
