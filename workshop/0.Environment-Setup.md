# Exercise 0: Environment Setup

> **Time:** ~20 minutes
> **Standalone:** Start here before any other exercise.

## Goal

Clone the Help Desk application, create a Python virtual environment, verify the app runs, and install the GitHub Copilot CLI extension.

---

## Steps

**1.** Verify required tools are installed:

```powershell
python --version   # 3.11+
git --version
gh --version
```

**2.** Authenticate with GitHub CLI:

```powershell
gh auth login
```

Follow the browser prompts. Confirm with:

```powershell
gh auth status
```

**3.** Clone the Help Desk application:

```powershell
gh repo clone CanarysPlayground/Help_Desk_Application
cd Help_Desk_Application
```

**4.** Create and activate a virtual environment, then install dependencies:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

> If you see an execution policy error: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

**5.** Start the application and verify it runs:

```powershell
uvicorn app.main:app --reload
```

Open http://localhost:8000/health — expect `{"status":"ok"}`. Press `Ctrl+C` to stop.

**6.** Install the GitHub Copilot CLI extension:

```powershell
gh extension install github/gh-copilot
gh copilot --version
```

**7.** Launch Copilot CLI chat to confirm it works:

```powershell
gh copilot chat
```

Type `exit` or press `Ctrl+C` to quit.
